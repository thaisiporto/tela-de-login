O que é esse projeto?

Esse projeto foi desenvolvido para Módulo 4 do Curso HTML5 e CSS3 do Curso em Vídeo, ministrado pelo professor Gustavo Guanabara. Projeto base desenvolvido pelo professor.

O tema do projeto foi escolhido pela autora e não possui nenhuma associação oficial com a plataforma de streaming Crunchyroll. As imagens são propriedade da própria plataforma, utilizadas nesse projeto sem fins comerciais, com o objetivo único de estudo.
